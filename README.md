# Калькулятор СНУ 

Калькулятор, использующий численные методы для решения систем нелинейных уравнений 

## Описание проекта

Программа для решения систем нелинейных уравнений с графическим или текстовым интерфейсом.

## Возможности 

- Решение СНУ методом половинного деления, Ньютона и хорд
- Графический интерфейс для удобного ввода данных
- Визуализация результатов
- Проверка корректности решения
